# Eye Care Appointment Booking System - Interview Assignment

## 📋 Overview

Full-stack TypeScript application for managing eye care appointments. This is a coding interview assessment for mid-level engineer candidates.

## 📦 What's Included

- **Frontend**: React 19 + TypeScript + Vite + Ant Design
- **Backend**: Node.js + Express 5 + TypeScript REST API
- Pre-configured testing setup (Vitest + Jest)
- Seed data for appointments, clinics, opticians, and services

## 🚀 Quick Start

1. **Setup Backend**

   ```bash
   cd backend
   npm install
   npm run dev  # Runs on http://localhost:3001
   ```

2. **Setup Frontend**
   ```bash
   cd frontend
   npm install
   npm run dev  # Runs on http://localhost:3000
   ```
